I. Settings

General Setting

a. Web site title: Set your web site title on the Internet browser.

b. Logo Image Src: Set your company logo image file. e.g. images/logo.gif (relative to your web root directory).

c. Check-Out Process: The Options when your customer check-out from their shopping cart.

  • Ask Customer Information: Check this box if you want your customer to enter their name, address, and phone information.
  • Ask Shipping Information: Check this box if you will ship your product to your customer.
  • Ask Billing Information: Check this box if you require your customer to enter their credit card number and billing address.
  • Show Review Order Page: Check this box if you want to show the "Review Order" page to your customer.

d. Email Messages: The options if you want to carbon copy the automated email messages to you.

  • Send the copy of customer invoice email to you: carbon copy the customer invoice email to you.
  • Send the copy of user registration email to you: carbon copy the user registration email to you.
  • Send the copy of shipped order email to you: carbon copy the shipped order email to you.

e. E-Store components: The options if you want turn on/off some of your online store features.

  • Show User Account: Check this box if want to turn on the user account feature.
  • Show Wish List: Check this box if you want to turn on the wish list feature.

 

Themes Setting

Using the themes setting, you will be able to change the look and feel of your e-store web site within seconds. Using this setting you can also control the color and fonts that will be used on your site.

Theme's Properties:

Each theme doesn't always have the same number of property. However, they have similar property name with different number (e.g. bar 1 color, bar 2 color, etc.). The color in this theme property is using the HTML predefined color name (e.g. blue, black, etc.) or the HTML Hex Code (e.g. #FFFFFF for white or #000000 for black).

  • Themes: The type of themes you want to select (e.g. classic 2, outlook, techno, etc.)
  • Color Schemes: The color schemes for your theme that you'd like to select (e.g. aqua, blue, etc.)

Advance Setting:

Using the advance setting options, you could change the colors of your store themes which are not available on the color schemes. The following is the advance setting properties.

  • Bar [#] Color: The color of the theme's bar (if there are any bars).
  • Active Link Color: The color of the url link after it's been actived or clicked (usually it's purple and underline).
  • Link Color: The color of the url link before it's clicked (usually it's blue and underline).
  • Visited Link Color: The color of the url link after it's been visited or clicked (usually it's purple and underline). The difference between active link and visited link is visited appear after the Internet browser is opened or in session. However, active link only appear when you open the Internet browser at the first time.
  • Body BGColor: The color of your web site background.
  • Font Face: The type of font that is used throughout your web site.
  • Font Size: The size of font that is used throughout your web site.
  • Font Color: The color of font that is used throughout your web site.
  • Tab Active Color: The color of the menu tab when it's active or on focus (if there are any menu tabs on the selected theme).
  • Tab Inactive Color: The color of the menu tab when it's not active or out of focus (if there are any menu tabs on the selected theme).
  • Tab Font Face: The type of the font that is used on the menu tab (if there are any menu tabs on the selected theme).
  • Tab Font Size: The size of the font that is used on the menu tab (if there are any menu tabs on the selected theme).
  • Tab Font Color: The color of the font that is used on the menu tab (if there are any menu tabs on the selected theme).
  • Table Header Background: The color background of the table header.
  • Table Header Font Face: The type of the font on the table header
  • Table Header Font Size: The size of font on the table header
  • Table Header Font Color: The color of the font on the table header
  • Table Background: The background color on the table body
  • Icon Color: The color of icon such as shopping cart, wish list, etc.

 

Payment Services Setting

Using Payment Services Setting, you can configure the online payment gateways provider that has been linked to your e-store to accept credit card payment from your customer (e.g. PayPal or VeriSign). Our e-store currently accept 3 different payment gateway products which are PayPal, VeriSign PayFlow Link, and VeriSign PayFlow Pro.

Payment Services Properties:

Payment Gateways: select which payment gateway do you want to use with your e-store.

Manual

Using the manual options you will be able to process your customer credit card number manually.

PayPal

  • PayPal email account: PayPal is using your email address as your account username.

VeriSign PayFlow Link

  • Method:What type of payment method you want to accept.
  • Transaction Type: VeriSign transaction type. Please read your VeriSign PayFlow Link User Guide to get more detail explanation for each transaction type.
  • VeriSign User ID: Your VeriSign user id.
  • VeriSign Partner ID: Your VeriSign Partner ID (usually it's VeriSign).
  • Email Customer: Whether or not you want VeriSign to email your customer regarding their recent purchase on your store.
  • Show Confirmation: Whether or not you want VeriSign to show the confirmation page to your customer before processing the transaction.

VeriSign PayFlow Pro

  • Transaction Mode: Set your transaction mode for testing or live transaction.
  • Tender Type: What type of payment method you want to accept on your e-store.
  • Transaction Type: VeriSign transaction type. Please read your VeriSign PayFlow Pro User Guide to get more detail explanation for each transaction type.
  • VeriSign User ID: Your VeriSign user id.
  • VeriSign Password: Your VeriSign password.
  • VeriSign Vendor ID: Your VeriSign vendor id (usually it's the same as your VeriSign user id).
  • VerSign Partner ID: Your VeriSign partner id (usually it's VeriSign).

 

Shipping Setting

Using the shipping setting you can determine to use our UPS automatic shipping rate calculation or you have the option to enter the shipping rate manually.

Shipping Properties

  • Shipping Mode: Select auto or manual shipping rate calculation.
  • Shipping Method: Select the shipping method that you want to provide to your customer.
  • Shipping Origin Zip Code: Enter the zip code where your product will be shipped from.

 

Messages Setting

Using the messages setting, you can customize all the messages that will be sent/shown to your customer such as user account registration, customer invoice email, etc. In addition you can use our pre-defined dynamic variable in your custom message such as your customer first name, address, etc.

Messages Properties

  • Message Type: The type of message that you want to customize.
  • From: The from email address such as sales@yourcompany.com
  • Subject: The email subject that will appear when your customer receive the automatic email that will be sent from your web site.
  • Message: Enter your custom message that you want your customer to see in this text box field.

Note: Only the email type messages that will have the "From" and "Subject" properties.

 

Account Info Setting

Using Account Info setting, you can edit your account information with us. So, if you need change address or billing information such as credit card, etc. You don't have to email us. Just simply edit your client and billing information from this setting.

III. Manage Store

Manage Store Components

A store component is the component of your e-store site. It could be the list of your products or your own customize contents. Through the store components management you will be able to add, change/edit, and remove the e-store components of your eCommerce site. This is the unique feature of our product because we don't dictate you what components that should be in your e-store and how it should be layout on your e-store site. You determine what and where to put the components. In addition, you have choices to use our pre-built components or import your own components to your e-store. The components could be html, java, or PHP script components. The best thing is all the change you make will happen in real time meaning the changes reflect to your e-store instantaneously. You will see how flexible is our product through this store components management.

Add/Edit/Delete Store Components

Adding or editing store componentsis very easy. Just simply click on "Add New Component" button to add new component and click on "Edit" or "Delete" button beside each component within the table to edit or remove the existing component.

Store Components Properties
  • Built-in Components: These are pre-built components that we have setup for you. Currently there are 6 built-in components as follow:
    1. Browse: Show all product main categories and sub categories 1 in one single frame.
    2. Login: Show your customer user account login (only display if you have user account components on your e-store site).
    3. Search: Show the search engine component, so user can search your product catalog (This feature comes handy if you select the theme that doesn't come up with search engine by default).
    4. All Products: Show all products that you enter into the database.
    5. New Items: Show all new products.
    6. Used Items: Show all used products.
    7. Refurbished Items: Show all refurbished products.
    8. Product Group: Show the product group component that you have created.
  • Product Group: Select the product group component that you want add to your store. This option appear only if you select the product group component.
  • Title: The title of your store component.
  • Filename: the filename of your store component, e.g. sponsor.htm (it will be relative to your web directory) or you can put the whole links such as http://www.mycompany.com/sponsor.htm. This option only appear if you import an external component.
  • Type: Whether or not you want to put border around your component. The choice is frame or no frame.
  • Position: Where do you want to put your components (left, center or right). Any components that put on the center will have the small image and short description along with it. However, any components that put on the left or right section will only have the link.
  • Sequence: Set the order of your components. For example if you set the "new release" component to 1 and "for sale" component to 2, the "new release" component will appear before for "sale component" and vice versa. However if you leave it in blank or put the same number for all or some of the components, the order will appear randomly.
  • Component Type: Set your component type whether this content is your own custom content or SuryadiSoft built-in content.
  • Category: Set where do you want your component to appear in what main category. For example, if you want your component to appear in all category menu tab, set it to "All Category". If you want your component to appear only on Home menu tab, set it to "Home".
  • Style: Select the style of your component. This option appear if you choose the built-in component.

Manage Links
To create url links from your e-store site, you need to go to Manage Web Content link on the left menu of control panel and click on "Manage Links" button on Manage Web Content page. Click on "Add New Link" button to add a new link and "Edit" or "Delete" button on the table to edit or delete your existing link. Through this link, you can add your own page such as about your company page, your customer service information, store policy page, etc.
Category Properties:
  • link type: Type of the link whether it's text or image.
  • link text: The text name of the link.
  • link img src: If you choose the link type to image, you need to enter the image source file here, e.g. images/help.png (it will relative to your web directory).
  • link url: The link filename or url, e.g. help.htm or http://www.yourstore.com/help.htm.
  • link position: The position of your link whether it's on top or bottom (depends on whether the theme support both position. Most of themes support only the bottom position) .
  • link target: Where do you want the link to be opened. There are 3 target options:
    1. Self: Your link page will blend together with your e-store design.
    2. Parent: Your link page will replace your e-store design.
    3. New Window: Your link page will appear in new Internet browser window.

 

Manage Products/Inventory

Adding your products/inventory is very easy. Just click on "Manage Products/Inventory" on the left menu of control panel. Then, click on "Add New Product" button on Manage Products/Inventory page. To edit or remove your existing product from the list, just simply click on "Edit" or "Delete" button on the table.
Product Properties:
  • product name: The name or title of your product that will appear on your e-store site.
  • product description: The description of your product. You can use HTML tag in this description field to format the appearance of your description text.
  • product isbn: The product indentification number. It's the bar code on most common product and the isbn number on books. This field is optional (you can leave it blank if you want).
  • product quantity: The number quantity of your product inventory in hand.
  • product retail price: The retail price of your product. This is optional. It comes handy to show how much saving that your customer get by purchasing the product from your store.
  • product price: The price of your product before taxes and shipping.
  • product image small: The image of your product in small size. Make sure that the small image of your product has measurement 75 pixels x 75 pixels. If your product image is bigger or smaller than this size, it will be stretched or squeze into this size.
  • product image medium:The image of your product in medium size. Make sure that the medium image of your product has measurement 255 pixels x 255 pixels. If your product image is bigger or smaller than this size, it will be stretched or squeze into this size.
  • product image large:The image of your product in large size. This image can be as big as you want.
  • product main category: The main category of this particular product.
  • product sub category 1: The first sub-category of this particular product.
  • product sub category 2: The second sub-category of this particular product.
  • product condition: The condition of this particular product. There are 3 options: new, used, or refurbished (factory remanufactured).
  • product color choices: The color choices of your product (separated by comma, e.g. red,green,blue). Your customer will be able to choose the product color they want to purchase.
  • product size choices: The size of your product (separated by comma, e.g. small,large,extra large). Your customer will be able to choose the product size (for example what size of T-Shirt they want to buy).
  • product other choices: Any choices that comes to your product that your customer can select (separated by comma, e.g. vanilla,jasmine,blueberry). This option will be handy if you offer choices with your product such as air freshner, clothing pattern, etc.
  • product weight: The weight of your product. This field is required because it's used to determine the shipping rate if you're using our automatic shipping calculation.
  • product length: The length of your product package.
  • product width: The width of your product package.
  • product height: The height of your product package.

 

Manage Product Group

To create a new product group, simply click "New Product Group" button. To edit or remove you existing product group, click the "Edit" or "Delete" button.

Product Group Properties:
  • group name: Your product group name as it appears as your store component title.

 

Add Product To Group

After adding a product group, you need to add some of your products to your product group or classify your product. To add products into your product group, simply click on "Add Product To Group" button.

Add Product To Group Properties

  • Product List: The list of all the products that you have been entered into the database.
  • <Product Group Name>: The list of all products that has been added into the particular product group.

 

Order Product In Group

Order Product In Group Properties

Manage Customer

To manage your customer, just click on "Manage Customer" on the left menu of control panel. Then, click on "New Customer" button on Manage Customer page. To edit or remove your existing product from the list, just simply click on "Edit" or "Delete" button on the table. Most of the time you don't need to add the customer manually. When your customer purchase your product, they will enter their customer information and you will be able to see it from this page. This page will only appear when you don't use PayPal as your Payment Gateways.
Customer Properties:
  • user id: Your customer user id when they register an account on your web site. However, you need the user account feature on your e-store site, so your customer can have their own user account.
  • customer first name: Your customer first name.
  • customer mi name: Your customer middle initial (if any).
  • customer last name: Your customer last name.
  • customer email: Your customer email address.
  • customer phone day: You customer day phone number.
  • customer phone evening: Your customer evening phone number.
  • customer fax: Your customer fax number (if any).
  • customer address 1: Your customer main address.
  • customer address 2: Your customer additional address information such as apartment number, suite number, etc.
  • customer city: The city where your customer live.
  • customer state: The state where your customer live.
  • customer zip: The zip/postal code where your customer live.
  • customer country: The country where your customer live.

If you have a user account feature on your e-store site, your customer will be able to access their user account and view and edit their customer infomation.

 

Manage Shipping Rate

You can set the shipping cost for each of your product by accessing the Manage Shipping Rate page from the left menu of control panel. Before entering the cost of your product shipping, the first thing that you need to do is adding your shipping vendor.
Shipping Vendor
Shipping Vendor is the shipping carrier that you use to ship your product to your customer. To add a new shipping vendor, simply click the "Add New Vendors" button. You can also edit or remove the shipping vendor by clicking the "Edit" or "Delete" button on the table.
Shipping Vendor Properties:
  • shipping vendor name: Your shipping vendor name such as UPS, FedEx, USPS, etc.
  • shipping vendor method: Your shipping vendor method such as UPS 2Day, FedEx Next Day, USPS Priority Mail, etc.

 

Shipping Rate

After setting-up your shipping vendor, now you can add the shipping rate for each of your product.

Shipping Rate Properties:

  • product name:Select the product name.
  • shipping vendor: Select the shipping vendor for this particular product.
  • shipping method: Select the shipping method for this particular product.
  • one item rate: Enter the shipping cost for sending 1 item.
  • additional item rate: Enter the shipping cost for sending each additional item.
  • state: Enter the state for this particular shipping cost if any.
  • city: Enter the city for this particular shipping cost if any.
  • zip: Enter the zip for this particular shipping cost if any.
  • country: Enter the country for this particular shipping cost if any.

 

Manage Sales Tax

If you have any sales tax for your product, you can set it from the Manage Sales Tax page. To add a new sales tax, simply click the "Add Sales Tax" button. To edit or remove the sales tax, simply click the "Edit" or "Delete" button on the table.

Sales Tax Properties:
  • sales tax tax: Select the state.
  • sales tax rate: Enter the percentage rate of the sales tax in decimal (e.g. if the sales tax in California is 8.5%, then you will enter 0.085).

 

Manage Transaction/Orders

Using the manage transaction/orders page, you can check your customer most recent order and view the transaction history of your e-store. In addition, you will be able to update the status of your customer order as well as the shipping tracking number (if any).

To view the transaction/order detail, click on the url link on transaction id.

To update your customer purchase order status (whether it's shipped or back-ordered), click on the url link on transaction id, then update the status from the Purchase Info section.

 

Manage User

If you have the user account feature on your e-store, you will be able to manage your customer user account. This feature will allow your customer to check the status of their order from your web site. Usually you don't have to add a new customer manually because your customer can sign-up for a user account from your web site by themself. However, in case you need to add a new customer manually, you can do it by clicking on "New User" button. To edit or remove user, click on "Edit" or "Delete" button on the table.
User Properties:
  • user id: Your customer user id.
  • user password: Your customer user password.
  • user email: Your customer user email.
  • user first name: Your customer first name.
  • user last name: Your customer last name.

 

Manage Shopping Cart

If you have the user account feature on your e-store, you will be able to control the shopping cart of your e-store. Sometimes your customer add items on their shopping cart and never check it out. However, everytime they add items to the shopping cart, your product quantity will automatically decrease. Therefore, you might want to take action either email your customer to check out their items in the shopping cart or you can delete it by yourself. If you delete the item in the shopping cart, your product quantity will automatically increase. To remove a single item, you can simply click on "Delete" button. To empty all your customers shopping cart, you can just simply click on "Empty Shopping Cart" button.

 

Manage Wish List

If you have the user account and wish list feature on your e-store, you will be able to control the wish list of your e-store. To remove a single item from your customer wish list, you can simply click on "Delete" button. To empty all your customers shopping cart, you can just simply click on "Empty Wish List" button.
