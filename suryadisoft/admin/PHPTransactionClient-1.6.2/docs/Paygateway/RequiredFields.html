PHP Transaction Client v1.6.1 API Specification

Required Fields

Authorization | Capture | Credit | Sale | Void  (Accessed via the doTransaction() method)

Settle | Purge | Totals  (Accessed via the doBatchTransaction() method)

Minimum required fields for an AUTH transaction:
Field Setter method
account token setAccountToken()
credit card number setCreditCardNumber()
expire month setExpireMonth()
expire year setExpireYear()
charge type setChargeType(AUTH)
charge total setChargeTotal()

Minimum required fields for a CAPTURE transaction:
Field Setter method
account token setAccountToken()
charge type setChargeType(CAPTURE)
charge total setChargeTotal()
order number setOrderID()

Note - The order id and the capture reference id must be from the same authorization transaction.

Minimum required fields for a SALE transaction:
Field Setter method
account token setAccountToken()
credit card number setCreditCardNumber()
expire month setExpireMonth()
expire year setExpireYear()
charge type setChargeType(SALE)
charge total setChargeTotal()

Minimum required fields for a VOID transaction:
Field Setter method
account token setAccountToken()
charge type setChargeType(VOID)
order number setOrderID()

Minimum required fields for a VOID_AUTH, VOID_CAPTURE, VOID_CREDIT transaction:
Field Setter method
account token setAccountToken()
charge type setChargeType()
order number setOrderID()
reference ID setReferenceID()

Minimum required fields for a CREDIT transaction:
Field Setter method
account token setAccountToken()
credit card number setCreditCardNumber()
expire month setExpireMonth()
expire year setExpireYear()
charge type setChargeType(CREDIT)
charge total setChargeTotal()
order number setOrderID()

Minimum required fields for a CREATE_ORDER transaction:
Field Setter method
account token setAccountToken()
credit card number setCreditCardNumber()
expire month setExpireMonth()
expire year setExpireYear()
charge type setChargeType(CREATE_ORDER)
charge total setChargeTotal()

Minimum required fields for a CANCEL_ORDER transaction:
Field Setter method
account token setAccountToken()
order number setOrderID()

Minimum required fields for a CLOSE_ORDER transaction:
Field Setter method
account token setAccountToken()
order number setOrderID()

Minimum required fields for a SETTLE batch action:
Field Setter method
account token setAccountToken()
charge type setAction(SETTLE)

Minimum required fields for a TOTALS batch action:
Field Setter method
account token setAccountToken()
charge type setAction(TOTALS)

Minimum required fields for a PURGE batch action:
Field Setter method
account token setAccountToken()
charge type setAction(PURGE)
